# SplatDev.Reflection.Helpers

Reflection and attribute helper utilities for .NET applications. Provides convenience methods for reading custom attributes and performing reflection-based operations.

[![NuGet](https://img.shields.io/nuget/v/SplatDev.System.Reflection.Helpers)](https://www.nuget.org/packages/SplatDev.System.Reflection.Helpers)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

## Compatibility

| .NET Target | Package Version |
|---|---|
| net8.0 | 1.0.x |
| net10.0 | 1.0.x |

## Features

- `AttributeHelper` — Read and query custom attributes from types, properties, and members
- `ReflectionHelper` — General-purpose reflection utilities

## Installation

```bash
dotnet add package SplatDev.System.Reflection.Helpers
```

> **Note:** The NuGet package ID is `SplatDev.System.Reflection.Helpers`.
