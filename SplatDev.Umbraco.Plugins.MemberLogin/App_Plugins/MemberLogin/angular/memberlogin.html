Member Login Manager

Manage member login configuration and test authentication from the Umbraco backoffice.

    • Login form with remember me support
    • Forgot password with token-based reset
    • Account lockout handling
    • Approval workflow support

    API Endpoints

    • POST /umbraco/api/memberlogin/ResetPassword
