# SplatDev.Umbraco.DataTypes.USStates

US States data type for Umbraco — provides a pre-populated dropdown of all 50 states.
Targets Umbraco 13 (net8.0) and Umbraco 17 (net10.0).

## Installation

```bash
dotnet add package SplatDev.Umbraco.DataTypes.USStates
```

## Features

- Pre-configured data type with all 50 US states + DC
- Display as dropdown, checkbox list, or radio buttons
- Stores state abbreviation (e.g. "CA") with full name label

## Usage

After installation, the data type appears in the Umbraco backoffice under
**Settings > Data Types**. Add it to any document type as a property.
