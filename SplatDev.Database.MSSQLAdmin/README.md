# SplatDev.Database.MSSQLAdmin

MSSQL administration helpers for `SplatDev.Database`. Targets `net8.0` and `net10.0`.

## Installation

```bash
dotnet add package SplatDev.Database.MSSQLAdmin
```

## Features

- Backup and restore operations
- Database creation and schema initialization
- User and login management
- Index maintenance and statistics updates
