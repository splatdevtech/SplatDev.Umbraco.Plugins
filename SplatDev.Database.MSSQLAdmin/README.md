# SplatDev.Database.MSSQLAdmin

MSSQL database administration utilities extending `SplatDev.Database`. Provides SQL Server-specific management operations using `Microsoft.Data.SqlClient`.

[![NuGet](https://img.shields.io/nuget/v/SplatDev.Database.MSSQLAdmin)](https://www.nuget.org/packages/SplatDev.Database.MSSQLAdmin)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

## Compatibility

| .NET Target | Package Version |
|---|---|
| net8.0 | 1.0.x |
| net10.0 | 1.0.x |

## Features

- SQL Server database administration operations
- Extends the `SplatDev.Database` abstractions for MSSQL-specific tasks

## Installation

```bash
dotnet add package SplatDev.Database.MSSQLAdmin
```

## Dependencies

- Microsoft.Data.SqlClient 5.2.1
- SplatDev.Database (project reference)
