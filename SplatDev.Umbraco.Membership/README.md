# SplatDev.Umbraco.Membership

Membership extensions for Umbraco CMS. Simplifies member registration and authentication workflows.

[![NuGet](https://img.shields.io/nuget/v/SplatDev.Umbraco.Membership)](https://www.nuget.org/packages/SplatDev.Umbraco.Membership)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

## Compatibility

| Umbraco Version | .NET Target | Package Version |
|---|---|---|
| Umbraco 13.x | net8.0 | 2.0.x |
| Umbraco 17.x | net10.0 | 2.0.x |

## Features

- `RegisterExtensions` — Streamlined member registration helpers for Umbraco's membership system

## Installation

```bash
dotnet add package SplatDev.Umbraco.Membership
```
