# SplatDev.Routing

Convention-based route registration helpers for ASP.NET Core applications. Provides an `IRoute` interface and `RouteService` for automatic route discovery and registration.

[![NuGet](https://img.shields.io/nuget/v/SplatDev.Routing)](https://www.nuget.org/packages/SplatDev.Routing)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

## Compatibility

| .NET Target | Package Version |
|---|---|
| net8.0 | 1.0.x |
| net10.0 | 1.0.x |

## Features

- `IRoute` — Route definition contract
- `RouteService` — Automatic route scanning and registration

## Installation

```bash
dotnet add package SplatDev.Routing
```

## Dependencies

- Microsoft.AspNetCore.App (framework reference)
