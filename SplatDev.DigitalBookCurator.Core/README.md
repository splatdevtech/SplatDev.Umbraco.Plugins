# SplatDev.DigitalBookCurator.Core

Digital book curation engine — metadata extraction, format conversion, and cataloging.
Targets `net8.0` and `net10.0`.

## Installation

```bash
dotnet add package SplatDev.DigitalBookCurator.Core
```

## Features

- EPUB, PDF, MOBI metadata extraction
- Cover image extraction
- ISBN and DOI lookup
- Format validation and conversion
- Catalog indexing
