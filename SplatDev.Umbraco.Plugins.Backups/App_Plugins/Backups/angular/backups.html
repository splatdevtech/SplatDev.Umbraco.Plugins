Backup Schedule
Backup scheduling is configured via appsettings.json under the SplatDev:Backups:Schedule section. Changes below are for reference — apply them in your configuration file.
Format: minute hour day-of-month month day-of-week
appsettings.json snippet 
{
  "SplatDev": {
    "Backups": {
      "Schedule": {
        "Enabled": {{ vm.scheduleForm.enabled }},
        "CronExpression": "{{ vm.scheduleForm.cronExpression }}",
        "Scope": {{ vm.scheduleForm.scope }},
        "Compress": {{ vm.scheduleForm.compress }},
        "KeepLocal": {{ vm.scheduleForm.keepLocal }}
      }
    }
  }
}
