# SplatDev.Messaging.Mailgun

Mailgun email provider for the `SplatDev.Messaging` abstraction layer. Send emails through the Mailgun API.

[![NuGet](https://img.shields.io/nuget/v/SplatDev.Messaging.Mailgun)](https://www.nuget.org/packages/SplatDev.Messaging.Mailgun)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

## Compatibility

| .NET Target | Package Version |
|---|---|
| net8.0 | 1.0.x |
| net10.0 | 1.0.x |

## Installation

```bash
dotnet add package SplatDev.Messaging.Mailgun
```

## Dependencies

- mailgun_csharp 0.9.0
- SplatDev.Messaging (project reference)
