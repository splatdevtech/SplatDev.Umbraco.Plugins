Model / Property Annotations

Section 1 — Credentials:

  • OlistConfigModel.ApiToken — [NopResourceDisplayName("Plugins.Misc.Olist.ApiToken")] password field
  • OlistConfigModel.UseSandbox — [NopResourceDisplayName("Plugins.Misc.Olist.UseSandbox")] bool toggle (radio)

Section 2 — Sync Modules:

  • OlistConfigModel.SyncProducts, .SyncOrders, .SyncCustomers, .SyncStock — bool switches

Section 3 — Manual Sync:

  • POST /Admin/OlistConfigure/ManualSync?module={Products|Orders|Customers|Stock}
  • POST /Admin/OlistConfigure/FullResync

Section 4 — Sync Logs:

  • GET /Admin/OlistConfigure/GetLogs?page=1&pageSize=50 → PagedList<SyncLogEntry>
  • SyncLogEntry: Timestamp, Module, Direction, RecordCount, Status, Message
