# SplatDev.Payments.Stripe

Stripe payment provider for the `SplatDev.Payments` abstraction layer. Process payments globally using the Stripe API.

[![NuGet](https://img.shields.io/nuget/v/SplatDev.Payments.Stripe)](https://www.nuget.org/packages/SplatDev.Payments.Stripe)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

## Compatibility

| .NET Target | Package Version |
|---|---|
| net8.0 | 1.0.x |
| net10.0 | 1.0.x |

## Installation

```bash
dotnet add package SplatDev.Payments.Stripe
```

## Dependencies

- Stripe.net 46.2.1
- Newtonsoft.Json 13.0.3
- SplatDev.Payments (project reference)
