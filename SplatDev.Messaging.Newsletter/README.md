# SplatDev.Messaging.Newsletter

Newsletter and bulk email provider for the `SplatDev.Messaging` abstraction layer. Implements bulk messaging operations for newsletter distribution.

[![NuGet](https://img.shields.io/nuget/v/SplatDev.Messaging.Newsletter)](https://www.nuget.org/packages/SplatDev.Messaging.Newsletter)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)
