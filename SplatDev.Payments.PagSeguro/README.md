# SplatDev.Payments.PagSeguro

PagSeguro payment provider for the `SplatDev.Payments` abstraction layer. Integration with the PagSeguro payment gateway for Brazilian e-commerce.

[![NuGet](https://img.shields.io/nuget/v/SplatDev.Payments.PagSeguro)](https://www.nuget.org/packages/SplatDev.Payments.PagSeguro)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

## Compatibility

| .NET Target | Package Version |
|---|---|
| net8.0 | 1.0.x |
| net10.0 | 1.0.x |

## Installation

```bash
dotnet add package SplatDev.Payments.PagSeguro
```

## Dependencies

- Newtonsoft.Json 13.0.3
- RestSharp 112.1.0
- SplatDev.Payments (project reference)
