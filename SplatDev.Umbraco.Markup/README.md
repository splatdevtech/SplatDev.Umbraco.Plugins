# SplatDev.Umbraco.Markup

HTML and markup helpers and extensions for Umbraco CMS. Provides Razor view extensions, media helpers, grid layout utilities, and HTML content manipulation.

[![NuGet](https://img.shields.io/nuget/v/SplatDev.Umbraco.Markup)](https://www.nuget.org/packages/SplatDev.Umbraco.Markup)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

## Compatibility

| Umbraco Version | .NET Target | Package Version |
|---|---|---|
| Umbraco 13.x | net8.0 | 2.0.x |
| Umbraco 17.x | net10.0 | 2.0.x |

## Features

- `GridExtensions` — Umbraco Grid layout rendering helpers
- `HtmlExtensions` — HTML content generation and manipulation
- `MediaExtensions` — Media item URL and property helpers
- `PublishedContentExtensions` — Content rendering extensions
- `PublishedContentPropertyExtensions` — Property value rendering
- `RazorExtensions` — Razor view helper methods
- `TagType` — HTML tag type enumeration

## Installation

```bash
dotnet add package SplatDev.Umbraco.Markup
```

## Dependencies

- Newtonsoft.Json 13.0.3
- Microsoft.AspNetCore.App (framework reference)
