# SplatDev.Messaging.SMSTools

SMSTools SMS provider for the `SplatDev.Messaging` abstraction layer. Send SMS messages through the SMSTools API.

[![NuGet](https://img.shields.io/nuget/v/SplatDev.Messaging.SMSTools)](https://www.nuget.org/packages/SplatDev.Messaging.SMSTools)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

## Compatibility

| .NET Target | Package Version |
|---|---|
| net8.0 | 1.0.x |
| net10.0 | 1.0.x |

## Installation

```bash
dotnet add package SplatDev.Messaging.SMSTools
```

## Dependencies

- RestSharp 112.1.0
- SplatDev.Messaging (project reference)
